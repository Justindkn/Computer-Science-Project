How to... Big Project?

Project Overview

At the beginning of this period of expeditions we were asked to design and carry out a project that involved our knowledge of HTML, CSS, and JS. I decided to make a "text-to-letter-design-maker-thing." Basically it takes letters you type into a text area and turns it into... say for example... block letters. On this "How to" page I will talking about how my website does the "text-to-letter" part.

The HTML

For the HTML, basically what I did was create an input box where you can type in text with a textarea tag. I then create dropdowns for font size and text design choice. At the bottom I created another textarea field that would display the transformed words.

The Javascript

The javascript of my Big Project wasn't overly complicated, but it was repetitive.

if(e.keyCode === 32){ output1.innerHTML += ""; }

else if(e.keyCode === 190){ output1.innerHTML += "";}

First I add an event listener "keyup" to make it so this function will only activate when a keyboard key is pressed. Then I put a bunch of if/else statements to tell the function to perform a certain action upon a certain key being pressed. I did this using keyCodes. A keyCode is a number that corresponds to a key on the keyboard. For example, the spacebar is keyCode 32. The function performed when the appropriate keyCode was pressed would insert the proper image into the output area. It does this with innerHTML. Upon being pressed, it takes the value of inpt 2 as the height of the image. Then it finds the value of dropDown(the font style chosen), adds an underscore, adds the name of the text, and adds .PNG. The complete name would be, for example, Block_A.PNG.
